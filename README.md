# cucumber-course

This is a sample Cucumber project for QA Course, demonstrating test automation.

## Requirements

- NodeJS
- Grunt CLI
- Selenium
- Cucumber

## Installation

### Windows

    $ install.bat
    
### Linux

    $ sh install.sh
    
### Manual

    $ npm i -g grunt-cli
    $ npm i -g selenium-webdriver
    $ npm i -g cucumber
    $ webdriver-manager update
    $ npm i
    
## Running tests

    $ grunt
