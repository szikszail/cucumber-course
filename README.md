# cucumber-course

This is a sample Cucumber project for QA Course, demonstrating test automation.

## Requirements

- NodeJS
- Grunt CLI
- Selenium
- Cucumber

## Installation

### Windows

    $> install.bat
    
### Linux

    $> sh install.sh
    
### Manual

    $> npm i -g grunt-cli@0.1.13 selenium-webdriver@2.47.0 cucumber@0.7.0
    $> webdriver-manager update
    $> npm i
    
## Running tests

    $> grunt
