# cucumber-course

[![dependency Status](https://david-dm.org/szikszail/cucumber-course.svg)](https://david-dm.org/szikszail/cucumber-course)

This is a sample Cucumber project for QA Course, demonstrating test automation.

## Requirements

- NodeJS
- Grunt CLI
- Selenium
- Cucumber

## Installation

### Windows

    $> install.bat
    
### Linux

    $> sh install.sh
    
### Manual

    $> npm i -g grunt@1.0.1
    $> npm i
    
## Running tests

    $> grunt
